rong-im-demo是基于融云通讯服务开发的Android版IM工具，实现了维护用户信息的服务端。目前有聊天、登录、通讯录等基础功能，方便集成和二次开发。

## 项目架构

参考融云的[结构介绍](https://docs.rongcloud.cn/v4/5X/views/im/noui/intro/structure.html)
